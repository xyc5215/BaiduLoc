# 百度网络定位工具 (BDLoc)

一个基于百度网络定位接口开发的 Android 原生定位工具，支持一键获取当前网络定位信息，适配 AIDE Pro 直接编译运行。

## 功能特性
- ✅ 完全复现百度网络定位接口请求，精准获取位置信息
- ✅ 自动解析 JSON 结果，直观展示省市区详细地址、经纬度、定位精度
- ✅ 支持一键刷新定位，实时更新位置数据
- ✅ 纯原生 Java 开发，无第三方依赖，AIDE Pro 直接编译零报错
- ✅ 适配 Android 8.0+ 系统，兼容低版本设备

## 接口说明
本工具基于百度公开网络定位接口 `https://loc.map.baidu.com/sdk.php` 开发，通过模拟移动端请求实现网络定位，无需申请百度地图开发者密钥。

## 编译与运行
### 环境要求
- AIDE Pro 编辑器（Android 端）
- Android 8.0+ 设备

### 编译步骤
1.  将项目导入 AIDE Pro，确保包名为 `com.xyc5215.bdloc`
2.  确认 `AndroidManifest.xml` 已添加网络权限
3.  直接点击编译，生成 APK 安装即可

### 权限说明
- `INTERNET`：用于发起网络定位请求
- `ACCESS_NETWORK_STATE`：用于获取网络状态，辅助定位


## 效果展示
- 打开 APP 自动发起定位，展示解析后的详细地址、经纬度、定位时间
- 点击「刷新定位」按钮，可重新获取最新位置信息
- 保留原始 JSON 数据，方便调试与二次开发

## 注意事项
- 本工具仅用于学习交流，请勿用于非法用途
- 定位精度依赖网络环境，Wi-Fi 环境下定位更精准
- 接口参数 `bloc` 为设备相关信息，可根据实际设备调整
